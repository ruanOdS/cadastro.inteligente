📋 Cadastro Inteligente

Projeto desenvolvido como atividade prática de Desenvolvimento Front-End, com foco na criação de formulários modernos, responsivos e com validação em tempo real.
🚀 Objetivo

Construir um sistema de cadastro de usuários com melhor experiência, reduzindo erros e melhorando a usabilidade.

🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (Flexbox e responsividade)
- JavaScript (validação em tempo real)
- Git e GitHub

💡 Funcionalidades

✔ Formulário de cadastro completo  
✔ Validação de campos em tempo real  
✔ Feedback visual (cores de erro e sucesso)  
✔ Confirmação de senha  
✔ Layout responsivo (mobile e desktop)  
✔ Simulação de envio com loading  

🔒 Regras de Validação

- Nome: mínimo de 3 caracteres  
- Email: formato válido obrigatório  
- Senha:
  - mínimo 8 caracteres  
  - pelo menos 1 letra maiúscula  
  - pelo menos 1 número  
- Confirmação de senha deve ser igual à senha
